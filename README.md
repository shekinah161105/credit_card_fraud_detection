# Credit Card Fraud Detection System

This project is a Machine Learning based web application that detects whether a credit card transaction is **Fraudulent** or **Legitimate**.

## Project Description
Online transactions are increasing rapidly and fraud detection is important.  
This system uses a **Decision Tree Machine Learning model** to predict whether a transaction is safe or fraudulent.

The application is built using **Python Flask** and provides a simple web interface where users can enter transaction details and check the result.

## Technologies Used
